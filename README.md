# OBELISK-Z

**Secure Wallet Control Layer for the ZETTA Ecosystem**

OBELISK-Z is the institutional interface for the ZETTA ecosystem's wallet infrastructure — combining multichain access, layered security evaluation, privacy-oriented operational modes and Zion-assisted intelligence in a unified sovereign environment.

## Architecture

```
├── public/images/           Optimized static assets
├── src/
│   ├── styles/              Modular CSS architecture
│   │   ├── variables.css    Design tokens
│   │   ├── base.css         Global styles & typography
│   │   ├── nav.css          Navigation system
│   │   ├── hero.css         Hero section
│   │   ├── sections.css     All content sections
│   │   ├── animations.css   Keyframe animations
│   │   └── responsive.css   Full responsive breakpoints
│   ├── scripts/
│   │   ├── i18n/            Internationalization (EN, PT, ES, ZH)
│   │   ├── canvas/          Animated visualizations
│   │   ├── security/        Security framework
│   │   ├── nav.js           Navigation logic
│   │   ├── reveal.js        Scroll animations
│   │   └── terminal.js      Terminal simulation
│   └── main.js              Application entry point
├── index.html               Semantic HTML
├── vite.config.js           Build configuration
└── vercel.json              Deployment configuration
```
